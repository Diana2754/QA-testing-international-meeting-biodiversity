# Bug Report

## ID
BUG-004

## Título
El logo del sitio no redirige a la página de inicio

## Severidad
Media

## Prioridad
Media

## Estado
Abierto

---

## Información del entorno

- Ambiente: Dev
- Dispositivo: Desktop
- Navegador: Chrome
- Fecha: 02/02/2025
- Tester: Diana Alali

---

## Descripción

El logo del sitio ubicado en el header no redirige a la página principal cuando se hace clic sobre él.

En la mayoría de los sitios web, el logo funciona como un acceso rápido al inicio, por lo que esta funcionalidad es esperada por los usuarios.

---

## Pasos para reproducir

1. Ingresar al sitio web del evento.
2. Navegar hacia cualquier sección del sitio.
3. Hacer clic sobre el logo ubicado en el header.

---

## Resultado esperado

El usuario debería ser redirigido a la página **Home** del sitio.

---

## Resultado actual

El logo no ejecuta ninguna acción al hacer clic.
