# Bug Report

## ID
BUG-001

## Título
El carrusel no funciona en dispositivos móviles

## Severidad
Alta

## Prioridad
Alta

## Estado
Abierto

---

## Información del entorno

- Ambiente: Dev
- Dispositivo: Mobile
- Resolución: 375px
- Navegador: Chrome
- Fecha: 02/02/2025
- Tester: Diana Alali

---

## Descripción

En la versión mobile del sitio del evento cultural internacional, el carrusel principal no permite navegar entre las diferentes imágenes.

Los botones de navegación no responden al intentar cambiar de slide.

Esto impide que el usuario pueda visualizar el contenido completo del carrusel.

---

## Pasos para reproducir

1. Ingresar al sitio web del evento
2. Visualizar el sitio en resolución mobile (375px)
3. Desplazarse hasta la sección del carrusel
4. Intentar cambiar de slide utilizando los botones de navegación

---

## Resultado esperado

El usuario debería poder navegar correctamente entre las distintas imágenes del carrusel.

---

## Resultado actual

Los botones de navegación no responden y el carrusel permanece en el mismo slide.
